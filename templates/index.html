Rejilla DWDM según ITU-T G.694.1

La recomendación ITU-T G.694.1 define una rejilla fija de frecuencias centrales para sistemas DWDM, basada en una frecuencia de referencia de 193.1 THz. A partir de esta referencia se generan los canales ópticos con una granularidad mínima de 12.5 GHz.

Las tablas siguientes muestran las frecuencias centrales nominales y sus longitudes de onda aproximadas para diferentes espaciamientos de canal. Debido a la gran cantidad de canales, las tablas se presentan de forma desplegable.
